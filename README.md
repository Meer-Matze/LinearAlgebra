# 线性代数笔记

这是一个现成的[LaTeX模板](https://zhuanlan.zhihu.com/p/604236564)编写的线性代数学习笔记项目。笔记涵盖了线性代数的基础概念、定理和应用，旨在整理所学线性代数知识。

## 概况
原本只是为了练习LaTeX才写的笔记，现在会过去看，发现~~怎么全是习题~~自己还写了蛮多的，于是就分享出来了

## 目录
- 第一章 行列式的计算
- 第二章 矩阵
- 第三章 向量空间与线性变换
- 第四章 线性方程组
- 第五章 相似对角化
- 第六章 二次型
- 第七章 线性代数综合练习
