# 线性代数笔记
[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)
---
这是一个现成的[LaTeX模板](https://zhuanlan.zhihu.com/p/604236564)编写的线性代数学习笔记项目。笔记涵盖了线性代数的基础概念、定理和应用，旨在整理所学线性代数知识。

## 概况
原本只是为了练习LaTeX才写的笔记，现在会过去看，发现<del>怎么全是习题</del>自己还写了蛮多的，于是就分享出来了

## 目录
- [第一章 行列式的计算](chap1/chap.tex)：行列式的定义、性质、克拉默法则及计算技巧。
- [第二章 矩阵](chap2/chap.tex)：矩阵运算、逆矩阵、分块矩阵及矩阵初等变换。
- [第三章 向量空间与线性变换](chap3/chap.tex)：向量组的线性相关性、秩、向量空间及线性变换。
- [第四章 线性方程组](chap4/chap.tex)：齐次与非齐次线性方程组解的结构与求解方法。
- [第五章 相似对角化](chap5/chap.tex)：特征值与特征向量、相似矩阵及实对称矩阵的对角化。
- [第六章 二次型](chap6/chap.tex)：二次型及其矩阵表示、合同变换、正定二次型。
- [第七章 线性代数综合练习](chap7/chap.tex)：全书知识点的综合回顾与练习。

## 如何编译
建议使用 **XeLaTeX** 编译器，环境配置推荐使用 `TexLive` 或 `MiKTeX`。
1. 确保安装了 [package.tex](config/package.tex) 下的宏包。
2. 编译主文件 [main.tex](main.tex)。
